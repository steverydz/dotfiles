# Dotfiles

These dotfiles are customised for me. You are welcome to use them but I do not offer any support for them.

## Packages to install

```
build-essential
curl
docker-compose
git
python3.6
python3-pip
python3-venv
tmux
vim
wget
zsh
xclip
```

## Install tmux package manager
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
